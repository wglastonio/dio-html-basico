Desafio HTML

Descrevendo tags html

<h1> a <h6>

Descritores de título, sendo h1 maior fonte e h6 menor fonte.

<p>

Delimitador de parágrafo para apresentação de texto.

<mark>

Destacar um texto.

<small>

Deixar um texto em tamanho menor.

<i>

Deixar um texto em itálico.

<u>

Deixar um texto sublinhado.

<strong>

Deixar um texto em negrito.

<ol>

Cria estrutura para uma lista ordenada.

<ul>

Cria estrutura para uma lista não ordenada.

<li>

Cria os elementos dentro de uma lista.

<a>

Cria estrutura para inserir hiperlinks.

<hr>

Cria uma linha horizontal separadora.

<sub>

Deixar um texto subescrito.

<sup>

Deixar um texto sobrescrito.

<font>

Utilizada para modificar o tamanho, a família/tipo e a cor da fonte.

<del>

Indica um texto que terá uma parte deletada do documento.

<abbr>

Apresenta uma abreviação para um texto maior ou uma sigla.
